PyTorch入门教程(1)——什么是PyTorch

Jun 3, 2017 2 min read

写了半年的TensorFlow，时常被tf一些奇怪的语法所困扰，特别是其没法直接打印tensor值的设计让我在调试网络时感到非常的不方便，于是打算转投PyTorch的阵营了。该系列关于PyTorch的极简使用说明主要翻译自其官方的入门教程Deep Learning with PyTorch: A 60 Minute Blitz

以下是本系列教程的第一篇什么是PyTorch

PyTorch的基于Python的一个用于科学计算的包，它主要有以下两个特点：

  • 可以看做是可以运行在GPU上的numpy
  • 为深度学习研究提供了极大地便利和效率

现在就让我们开始PyTorch的入门之旅。

Getting Started

Tensors

Tensors比较类似于numpy中的ndarrays，但是不同在于Tensors可以构建在GPU上利用其进行加速计算。 首先导入PyTorch

from __future__ import print_function
import torch

然后让我们创建一个5x3大小的未初始化的矩阵

x = torch.Tensor(5, 3)
print(x)

Out:
1.00000e-25 *
  0.4136  0.0000  0.0000
  0.0000  1.6519  0.0000
  1.6518  0.0000  1.6519
  0.0000  1.6518  0.0000
  1.6520  0.0000  1.6519
[torch.FloatTensor of size 5x3]

可以看到因为未初始化，所以创建的值基本都是0 创建一个随机初始化的矩阵

x = torch.rand(5, 3)
print(x)

Out:
0.2598  0.7231  0.8534
 0.3928  0.1244  0.5110
 0.5476  0.2700  0.5856
 0.7288  0.9455  0.8749
 0.6663  0.8230  0.2713
[torch.FloatTensor of size 5x3]

如果你要获取某个Tensor的size，只需要

print(x.size())

Out:
torch.Size([5, 3])

非常numpy的写法，事实上torch.Size就是一个元组，可以通过x.size()[0]获取x第一维的大小

操作

在PyTorch中有许多等效的运算符写法，这里以加法为例

写法1

y = torch.rand(5, 3)
print(x + y)

Out:
0.7931  1.1872  1.6143
 1.1946  0.4669  0.9639
 0.7576  0.8136  1.1897
 0.7431  1.8579  1.3400
 0.8188  1.1041  0.8914
[torch.FloatTensor of size 5x3]

写法2

print(torch.add(x, y))

Out:
0.7931  1.1872  1.6143
 1.1946  0.4669  0.9639
 0.7576  0.8136  1.1897
 0.7431  1.8579  1.3400
 0.8188  1.1041  0.8914
[torch.FloatTensor of size 5x3]

写法3，有一个输出的tensor

esult = torch.Tensor(5, 3)
torch.add(x, y, out=result)
print(result)

Out:
0.7931  1.1872  1.6143
 1.1946  0.4669  0.9639
 0.7576  0.8136  1.1897

如果想想实现+=的效果

# adds x to y
y.add_(x)
print(y)

Out:
0.7931  1.1872  1.6143
 1.1946  0.4669  0.9639
 0.7576  0.8136  1.1897
 0.7431  1.8579  1.3400
 0.8188  1.1041  0.8914
[torch.FloatTensor of size 5x3]

很多其他的方法加上都可以实现类似的效果，如x.copy(y), x.t_()等都会立即改变x的值。

跟TensorFlow中的tensor差别最大的可能就是PyTorch中的tensor可以直接进行索引操作，如

print(x[:, 1])

Out:
0.7231
 0.1244
 0.2700
 0.9455
 0.8230
[torch.FloatTensor of size 5]

这在TensorFlow中是没法直接做到了，这也是PyTorch最吸引我的地方了。

与Numpy之间进行转换

From tensor To numpy

如果想把一个tensor转换为一个numpy对象，只需要使用.numpy()即可

a = torch.ones(5)
print(a)

Out:
1
 1
 1
 1
 1
[torch.FloatTensor of size 5]

b = a.numpy()
print(b)

Out:
[ 1.  1.  1.  1.  1.]

当我们改变a的值时，b也会立即跟着改变

a.add_(1)
print(a)
print(b)

Out:
2
 2
 2
 2
 2
[torch.FloatTensor of size 5]

[ 2.  2.  2.  2.  2.]

From numpy To tensor

反过来，如果想把一个numpy数组转换成一个tensor也很容易

import numpy as np
a = np.ones(5)
b = torch.from_numpy(a)
np.add(a, 1, out=a)
print(a)
print(b)

Out:
[ 2.  2.  2.  2.  2.]

 2
 2
 2
 2
 2
[torch.DoubleTensor of size 5]

创建CUDA Tensors

PyTorch与numpy最大的不同就在于其可以在GPU上创建变量，方法很简单，只需要运行.cuda()就可以把一个tensor放入GPU中

# let us run this cell only if CUDA is available
if torch.cuda.is_available():
    x = x.cuda()
    y = y.cuda()
    x + y
